Executive Summary
ملخص تنفيذي للمشروع
EXEC-001

DPRIMS هو نظام ذكي متكامل لفحص السكك الحديدية في الوقت الحقيقي يعمل بالطاقة الشمسية، ويستخدم تقنيات الذكاء الاصطناعي والرؤية الحاسوبية لاكتشاف عيوب السكة أثناء حركة القطار دون توقف. يجمع النظام بين كاميرتين ESP32-CAM، ونموذج YOLOv8 للكشف الآني، وتحليل SSIM ثنائي الطور لمقارنة حالة السكة قبل وبعد مرور القطار، مع إرسال تنبيهات فورية للوحة تحكم React في حالة الخطر.

🎯

الهدف الرئيسي

الكشف التلقائي عن عيوب السكك الحديدية في الوقت الفعلي بدقة تتجاوز 90% باستخدام الذكاء الاصطناعي.

⚡

نظام طاقة شمسية متكامل يجعل القطار مكتفياً ذاتياً طاقياً دون الحاجة لمصدر خارجي.

🐍 Python AI Pipeline — الكود الأساسي

analysis.py — DPRIMS AI Core
# DPRIMS — AI Analysis Pipeline import cv2, requests, time, base64 from ultralytics import YOLO from skimage.metrics import structural_similarity as ssim # ─── Config ─────────────────────────────────────────────────── CAM_FRONT = "http://192.168.1.101/stream" CAM_REAR = "http://192.168.1.102/stream" BACKEND_URL = "http://192.168.1.100:3000/api/alert" SSIM_THRESH = 0.85 # below this = structural change detected # ─── Load Model ─────────────────────────────────────────────── model = YOLO("rail_defects_v3.pt") def analyze_frame_pair(front_frame, rear_frame, train_data): # 1. YOLOv8 inference on front frame results = model.predict(front_frame, conf=0.6, verbose=False) detections = results[0].boxes # 2. SSIM dual-phase comparison gray_front = cv2.cvtColor(front_frame, cv2.COLOR_BGR2GRAY) gray_rear = cv2.cvtColor(rear_frame, cv2.COLOR_BGR2GRAY) score, _ = ssim(gray_front, gray_rear, full=True) # 3. Severity classification severity = classify_severity(detections, score) # 4. Send alert if anomaly found if severity != "NORMAL": send_alert(detections, score, severity, train_data, front_frame, rear_frame) def classify_severity(detections, ssim_score): critical_classes = {"crack", "broken_rail", "rail_gap"} medium_classes = {"missing_bolt", "sleeper_damage"} for box in detections: label = model.names[int(box.cls)] if label in critical_classes: return "HIGH" if label in medium_classes: return "MEDIUM" if ssim_score < SSIM_THRESH: return "MEDIUM" return "NORMAL"

🖥️ تشغيل النظام الكامل

🔍 اكتشاف عيب LOW: مسمار ناقص

4

⚠️ اكتشاف عيب MEDIUM: شرخ صغير

YOLO يكتشف crack + SSIM Score = 0.76 (أقل من 0.85) → تنبيه برتقالي → توصية بتقليل السرعة

5

🚨 اكتشاف عيب HIGH: كسر كامل

broken_rail بثقة 0.93 → EMERGENCY STOP يُرسَل فوراً → القطار يتوقف → Dashboard يُظهر الموقع على الخريطة بدقيقة

6
